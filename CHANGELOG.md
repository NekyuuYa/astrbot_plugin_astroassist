# 更新日志 (CHANGELOG)

## [0.9.0] - 2026-04-03
### ✨ 核心特性
- **高德地图 API 集成**：支持通过地名直接设置位置，内置坐标纠偏逻辑（GCJ-02/BD-09 转 WGS-84），确保天文预报精度。
- **专业天文指标**：接入 7Timer! API，新增 **视宁度 (Seeing)** 与 **透明度 (Transparency)** 预报，提供 1-8 级专业分级参考。
- **日夜看板主题**：根据目标位置的日出日落时间，自动切换“浅色模式”与“深色模式”，夜间观测不再刺眼。
- **本地化超清渲染**：彻底弃用远程服务，改用本地 Playwright 引擎配合 3 倍超采样，输出 2100px 物理宽度的无损 PNG 原图。

### 🎨 视觉优化
- **Material Design 3 风格**：全中文化界面，采用 M3 标准 Tonal Palettes 配色与圆角卡片设计。
- **高密度排版**：
  - 看板宽度锁定为 780px，极致紧凑。
  - 日期列纵向合并，新增精准的“日出/日落”横向分割线。
  - 气温、风速、湿度、视宁/透明均采用 1:1 独立色阶方块。
- **云量拟真化**：云量方块采用蓝天/黑夜底色配合纯白/深灰填充，直观体现云层厚度。

### 🔧 稳定性与修复
- **指令系统重构**：完全基于消息监听器解析，自适应任意唤醒词，修复了因框架签名校验导致的指令失效报错。
- **环境自检系统**：新增 initialize 自动环境初始化与错误诊断，确保 Playwright 与内核依赖自动就绪。
- **数据降级逻辑**：当 7Timer! 接口异常时，相关指标自动填充为斜杠 `/` 并显示为灰色，确保插件整体不挂起。
